Detailed Description

Encapsulates the Csound 6 threading system for use by .net programs. Use this, instead of .net async/await patterns, if you wish to use csound threading (pthreads) exclusively. See Csound6PerformanceThread for an example of how to use this class. So that a C# or VB method can be used in a csound thread, a Run(object userdata) method is defined through the ICsoundRunnable interface. This is what the csound thread ultimatedly executes.
