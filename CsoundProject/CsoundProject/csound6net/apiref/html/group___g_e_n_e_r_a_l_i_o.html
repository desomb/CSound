void csound6netlib.Csound6Net.SetOutputFileName (string name, SoundFileType type, SampleFormat fmt)
 Sets the sound file name, type and format to use as samples are sent to disk during sound generation. Call this method to change the output file before calling ParseOrc or CompileOrc using strings for instrument definitions. CSD file specs which name an output file will supercede this call. Once Compile has been called, this method will be ignored internally by csound. The csound 6 API now uses strings to indicate type and format (it used to use integers). To reduce the likelihood of error, this method's enum maps to the full set of legal strings accepted by the "c"-level API.

Sets the sound file name, type and format to use as samples are sent to disk during sound generation. Call this method to change the output file before calling ParseOrc or CompileOrc using strings for instrument definitions. CSD file specs which name an output file will supercede this call. Once Compile has been called, this method will be ignored internally by csound. The csound 6 API now uses strings to indicate type and format (it used to use integers). To reduce the likelihood of error, this method's enum maps to the full set of legal strings accepted by the "c"-level API.

Parameters
nameFile name relative to SFDIR for sending sound samples
typeThe type sound file to write
fmtThe structure of individual samples
