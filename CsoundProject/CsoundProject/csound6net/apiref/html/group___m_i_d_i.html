async Task<IDictionary<string, CS_MIDIDEVICE> > csound6netlib.Csound6Net.GetMidiDeviceListAsync ( bool  isOutput)
inline

Gets this computer's input or output midi devices as visible to csound as a structure which is returned here in a dictionary keyed by device id. Callable with await keyword:

var outdevs = await csound.GetMidiDeviceListAsync(true);

Parameters
isOutputTrue to get a dictionary of midi output devices, false to get a dictionary of midi input devices
Returns
upon await, a dictionary of the available midi devices keyed by id
