Detailed Description

Represents parameter values which can be passed as flags to an instance of Csound6Net in order to influence how a performance will be executed. Csound exposes these parameters via the CSOUND_PARAMS structure which this class exposes via idiomatic C# property values. All of the values represented by this class are setable by input arguments from a command line.
