Processes a formated message using C#/.net string formatting rather than classic "c" formatting. Formatted string is merged in with the logging events along with those from the csound engine. Use this signature for default formatting.

This is different than the csound API because C# programmers (especially without a "c" background) would fine that awkward. Because messages from the .net host get folded in events just like messages from csound, there is no need for these messages to actually be presented to csound internally. /remarks>
